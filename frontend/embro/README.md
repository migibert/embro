# Requirements

* Node
* React
* Embro API

# Features

* Organization management
* Team management
* Skill management
* Collaborator management

# Configuration

The application can be configured throug environment variables

* EMBRO_API
* AUTH_AUDIENCE
* AUTH_DOMAIN
* AUTH_CLIENT_ID

# Running locally

Start the application

```
npm start
```

Package the application

```
npm run build
```

Run tests

```
npm test
```
